Install

$ curl -fsSL storozhenko98.github.io/tt/install.sh | bash copy
releases source

Play

$ tt # terminal 1
$ tt # terminal 2 (same LAN)
$ tt YourName # optional name

Players discover each other automatically via UDP broadcast. Select a player, Enter to challenge, Y to accept.

Controls

A / DMove paddle WToggle topspin SToggle backspin SpaceServe HHelp / rules QQuit

Spin

  • Topspin — ball accelerates, harder to return
  • Backspin — ball decelerates, deceptive
  • Sidespin — paddle movement at contact curves it
  • Angle — edge of paddle = sharper angle

How it works

  • UDP broadcast discovery on port 44144
  • Direct peer-to-peer, no relay server
  • Host-authoritative physics at 60fps
  • Braille-dot rendering for smooth motion
